# 🎬 MovieMate | Movie Ticket Booking App
### UX & Product Design Case Study

A mobile-first movie ticket booking application designed to simplify the ticket booking experience for busy users while making the product more accessible for first-time smartphone users and non-native English speakers.

---

## 📌 Project Overview

**Duration:** July 2022 – September 2022

This project was completed as part of the Google UX Design Professional Certificate. The objective was to redesign the movie ticket booking experience by identifying user pain points through research and validating design decisions through usability testing. 

---

## 🎯 Problem Statement

Existing movie booking experiences can be frustrating for:

- Busy working professionals who need a quick booking flow
- Users unfamiliar with mobile applications
- Non-native English speakers
- Users requiring accessibility support

The challenge was to create an intuitive, inclusive, and time-efficient booking experience. 

---

## 🎯 Goal

Design a mobile application that enables users to book movie tickets quickly while reducing friction through accessibility, simple navigation, and an intuitive booking flow.

# 🔬 User Research

Research included interviews and empathy mapping to understand how different users book movie tickets.

### Key Findings

- Working adults value speed over feature richness.
- Language barriers reduce usability.
- Accessibility needs are often overlooked.
- Users prefer minimal navigation with fewer steps.



---

# 😖 User Pain Points

## ⏱ Time

Users wanted to complete ticket booking in as few steps as possible.

## 🌍 Language

Many users preferred booking in their native language rather than English.

## ♿ Accessibility

Existing apps lacked accessibility features and inclusive design considerations.



---

**Allen**

# 🧪 Usability Testing

Two rounds of usability studies were conducted.

### Round 1

Users requested:

- Language selection earlier in the journey
- Better payment visibility
- More guidance throughout booking

### Round 2

Feedback led to:

- Simpler navigation
- Improved seat selection
- Better accessibility through color choices



---

# 🔄 Design Improvements

Major improvements after testing included:

✅ Bottom Navigation replacing the hamburger menu

✅ Simplified seat selection flow

✅ Improved booking experience

✅ Accessibility-focused color palette

✅ More intuitive navigation
